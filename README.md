## Example Usage

### Log a task/actvity unit

```
idid "brushed my teeth"
idid "walked the dog"
idid "drove to work"
idid "attended product meeting"
idid "code review"
idid "went to lunch"
```

### Print history

```
idid what
```
```
 07:24 brushed my teeth
 08:39 walked the dog
 09:02 drove to work
 10:46 attended product meeting
 12:13 code review
 14:01 went to lunch
```
